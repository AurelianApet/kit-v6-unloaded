2D Toolkit installation instructions:

1. In Unity, open the Assets/Scenes/ImportFrameworksHere scene.

2. Remove this TK2DROOT folder in Unity.

3. Import 2D Toolkit into Unity (everything will go into TK2DROOT). 

For full installation instructions, please read the installation page online: 
https://city-building-kit-with-pvp-battles.readme.io/v6.0/docs/installation
